class ibex_bluesky_core.fitting.FitMethod(model: Model | Callable[[ndarray[tuple[Any, ...], dtype[float64]]], ndarray[tuple[Any, ...], dtype[float64]]], guess: Callable[[ndarray[tuple[Any, ...], dtype[float64]], ndarray[tuple[Any, ...], dtype[float64]]], dict[str, Parameter]])[source]ÔÉÅ

Bases: object

Tell LiveFit how to fit to a scan. Has a Model function and a Guess function.

Model - Takes x values and a set of parameters to return y values. Guess - Takes x and y values and returns a rough ‚Äòguess‚Äô of the original parameters.

Assign model and guess functions.

Parameters:

  • model (lmfit.Model | Callable) ‚Äì The model function to use.

  • guess (Callable) ‚Äì The guess function to use.
